SOLVING PROBLEM USING  METHODS OF CONVERSION IN NUMBER SYSTEM :


1. CONVERT DECIMAL NUMBER - BINARY NUMBER : 

Let,

	 Decimal Number = (45)v10

		2  | 45	-- 1
	             -----
		2  | 22 	-- 0 
		     -----
		2  | 11	-- 1
		     -----
		2  |  5  	-- 1
		     -----
		2  |  2	-- 0
		     -----
		    |  1 	
 	
	Binary Number = (101101)v2

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2.  CONVERT DECIMAL NUMBER - OCTAL NUMBER :

Let,
	
	Decimal Number = (86)v10
	

		8  | 86	-- 6
	             -----
		8  | 10 	-- 2 
		     -----
		    |  1	

	Octal Number = (62)v8

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	Decimal Number = (92)v10

4. CONVERT BINARY NUMBER - DECIMAL NUMBER :

Let,

	Binary Number = (100110)v2

		
			1      0      0     1      1      0
		(x)  2^5  2^4  2^3  2^2  2^1  2^0
		----------------------------------------------
		        32 + 0 +  0  +  4  +  2  +  0 		= 38
		-----------------------------------------------

	Decimal Number = (38)v10

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
